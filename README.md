# Natural Africa - Flutter App
An offline Flutter app showcasing African natural resources.
This scaffold includes fully implemented screens, local JSON data, and offline favorites.

To run:
1. Install Flutter SDK.
2. Navigate to the project directory and run `flutter pub get`.
3. Run `flutter run`.

Notes:
- No image banners are included (per request).
- Sample data is minimal; extend `assets/data/countries.json` and `assets/data/resources.json`.
